# Формула Чистоты — лендинг

Легкий одностраничный сайт для клининговой компании «Формула Чистоты» (Чебоксары). Без сборщиков и зависимостей.

## Как запустить
Откройте `index.html` в браузере.

## Что заменить под реальные материалы
- Фото в `/assets/img/`:
  - `hero.jpg`
  - `gallery-1.jpg` ... `gallery-6.jpg`
  - `favicon.ico`
- Ссылку на WhatsApp в `index.html` (ищите TODO). Формат: `https://wa.me/79999999999`.
- Видео:
  - Замените ссылку YouTube в iframe на свой ролик **или** вставьте тег `<video>` с mp4.
- Отзывы: замените тексты на реальные.
- OG-мета теги (`og:url`, `og:image`) на актуальные.

## Структура проекта
```
/index.html
/assets/styles.css
/assets/script.js
/assets/img/   (папка для изображений)
/README.md
```
